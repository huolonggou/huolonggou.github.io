灭霸程序

　这几天看了复仇者联盟４,灭霸这个紫薯精虽然被灭了,不过给人的感觉还是很强的,打个响指宇宙间一半生命灰飞烟灭,忽然有点想写一个灭霸程序,双击打开输入路径后一半的文件灰飞烟灭.有点小酷
　我用的是python来实现这一程序,用的是os库,random库和shutil库.os库调用os.listdir()来列出某个目录下的所有文件,os.getcwd()获得程序所在目录,而os.remove负责删除文件,其实可以使用os.rmdir来删除目录,不过只能删除空文件夹,不够通用,于是使用shutil.rmtree来实现(既可以删除空目录,也可以删除包含文件和其他目录的父目录,比较方便.)利用random库来实现随机选择文件删除的目的,使用的是random.sample()函数来实现(从某一列表中随机选取k个元素组成新的列表)

顺便比较一下这几种删除函数

os.remove(),os.unlink(),os.removedirs(),shutil.rmtree()
os.remove()和os.unlink()这两个函数功能是一样的，都是删除一个文件,os.removedirs()是递归删除空文件夹比如E:\delete\tobedelete该目录下使用这个函数，如果delete中只有tobedelete这一个目录且delete是个空目录，则会先删除tobedelete然后删除delete目录，即先删除子目录再尝试删除父目录，如果目录是空的则进行删除操作，总体来说不好用，即使当作清理空文件夹的一个函数来使用，也清理不干净。不如使用shutil.rmtree()这个函数。这些函数的删除权限与用户权限相同，对于需要管理员权限才能删除的文件，需要以管理员身份运行脚本才有效，否则会拒绝访问。
